File Format Guide

Column Structure
  • Column A: Member name (must match system)
  • Column B+: Dates (DD-MM-YYYY format)
  • Data Cells: Shift codes (D, E, N, etc.)
Perfect For
  • Monthly shift schedules
  • Calendar-style planning view
  • Quick visual overview
  • Easy bulk updates
Important Notes
  • First column header must be "Member name"
  • Use DD-MM-YYYY date format
  • Names must exist in system
  • Use uppercase shift codes only

Shift Codes Reference

D Day Shift
E Evening Shift
N Night Shift
OS OnShore Shift
OF OffShore Shift
G General/Flexible
LE Leave
VL Vacation Leave
HL Holiday Leave
CO Comp Off
Pro Tips Leave cells blank for Off Days (no assignment). Shift codes are case-sensitive - always use UPPERCASE letters.
