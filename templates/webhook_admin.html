Power Automate Setup

Step 1 — Trigger

Add trigger: "When a new email arrives (V3)"
Filter subject to: INC (or your ServiceNow prefix)

Step 2 — Parse email body

Add "Initialize variable" actions to extract fields using these expressions:

  • incident_id — regex on subject: [A-Z]+\d+
  • assignment_group — parse line after "Assignment group:"
  • priority — parse line after "Priority:"
  • status — parse line after "State:" or "Status:"
  • assigned_to — parse line after "Assigned to:"
  • occurred_at — parse line after "Opened:" → ISO 8601
  • description — parse line after "Description:"
  • resolution_notes — parse line after "Resolution notes:"

Step 3 — HTTP action

  • Method: POST
  • URL: https://10.82.143.226/api/webhook/incidents
  • Header: X-Webhook-Token: <token above>
  • Header: Content-Type: application/json
  • Body: JSON with the variables from Step 2
