Configure Team Email Distribution Lists

Set up team-specific email recipients for handover notifications and priority alerts. Each team can have their own distribution list instead of using global configuration.

Configure Team Emails
How Team Email Configuration Works
  • Team-Specific Routing: Each team gets their own email notifications
  • Handover Recipients: Regular team members receive standard handover notifications
  • Priority Recipients: Additional recipients for high-priority/escalated incidents
  • Global Fallback: Teams without email config use global settings
