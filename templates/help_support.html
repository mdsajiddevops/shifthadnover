User Guide
Welcome to ShiftOps

ShiftOps is a comprehensive shift handover management system designed to ensure smooth transitions between shifts. Here's how to get started:

1
Login to the Application

Use your corporate SSO credentials (recommended) or username/password to sign in. SSO provides secure authentication with your EPAM account.

2
Navigate the Dashboard

The dashboard shows an overview of your team's current status including active incidents, open key points, and recent handover activities.

3
Explore Key Features

Use the sidebar menu to access Handover Form, Reports, Roster, Key Points, Change Info, and other tools.

Pro Tip: Use keyboard shortcuts for faster navigation. Press Ctrl+H for Handover, Ctrl+R for Reports.
Creating a Shift Handover
1
Select Shift Details

Choose the handover date, current shift type (Morning/Evening/Late Evening/Night/General Shift/OnShore/OffShore), and next shift type. The system auto-detects your scheduled shift.

2
Add Incidents

Document all incidents that occurred during your shift. Include incident type (Open/Closed/Escalated/Priority), title, description, and assigned engineer.

3
Record Key Points

Add important items that need attention. Key points carry forward until resolved. Include JIRA IDs for tracking.

4
Log Change Info

Record any changes, deployments, or maintenance activities with change numbers, descriptions, and status.

5
Save or Submit

Save as Draft: Save your progress and continue later.
Submit: Finalize the handover and send email notifications to the next shift team.

Note: Only engineers scheduled for the current shift can submit handovers. Admins can override this restriction if needed.
Viewing and Exporting Reports

The Reports section provides comprehensive views of all shift handovers with powerful filtering and export capabilities.

Available Report Types:
  • Handover Reports: Complete shift handover records
  • Change Info Reports: All change/deployment activities
  • KB Update Reports: Knowledge base updates
  • Incident Summary: Incident analytics by type/priority
  • Key Points Status: Open/closed key points tracking
Export Options:
CSV Excel PDF

Click the export buttons in the filter section to download reports in your preferred format. Exports respect your current filter selections.

Pro Tip: Use date range filters to generate weekly or monthly summary reports for management reviews.
Managing Shift Roster
Roster Features:
  • View Monthly Roster: See team schedules in calendar view
  • Team Availability: Check who's available on any date
  • Shift Allowance Report: Generate allowance reports for payroll
  • Upload Roster: Bulk upload from Excel files
Shift Codes:
D - Day E - Evening N - Night G - General VL - Vacation Leave CO - Comp Off
Uploading Roster:
  1. Download the roster template from the upload page
  2. Fill in team member names and shift codes
  3. Upload the Excel file
  4. Review and confirm the imported data
Key Points Management

Key Points are important items that need to be tracked across shifts until resolved. They automatically carry forward in handover forms.

Key Point Statuses:
Open In Progress Closed
Adding Updates:
  1. Go to Key Points page from the Operations menu
  2. Click on a key point to expand it
  3. Use the "Post Update" button to add progress notes
  4. Updates are timestamped and attributed to the user
Pro Tip: Add JIRA IDs to key points to link them with your project tracking system.
Admin Features

Admin users have access to additional configuration and management features.

Available for Team Admins:
  • Team Members: Add/edit team member details
  • Roster Upload: Upload and manage team rosters
  • Escalation Matrix: Configure escalation contacts
  • Vendor Details: Manage vendor contact information
Available for Account/Super Admins:
  • User Management: Create/edit user accounts
  • Email Configuration: Configure email settings
  • Email Monitoring: Track email delivery status
  • System Configuration: Manage application settings
  • Active Sessions: Monitor logged-in users
  • Audit Logs: View user activity history
Super Admin Only:
  • Delete Draft Reports: Remove draft handovers from reports page
  • Manage All Accounts: Full access across all organizations
Frequently Asked Questions
General Questions

To create a new shift handover:

  1. Click "Handover" in the sidebar menu
  2. Select the handover date and shift types (current → next)
  3. Add incidents, key points, change info, and KB updates
  4. Click "Save as Draft" to save progress, or "Submit Handover" to finalize and send emails

The system will automatically populate current shift engineers based on your team's roster.

The system validates that only engineers scheduled for the current shift can submit handovers. This ensures accountability and accuracy.

Solutions:

  • Verify you're listed in the current shift engineers dropdown
  • Check if your roster is correctly uploaded for today
  • Make sure you selected the correct shift type (Morning/Evening/Late Evening/Night/General Shift)
  • Contact your admin if you need to submit outside your scheduled shift

Go to Reports → Handover Reports in the sidebar menu.

Features:

  • Filter by date, shift type, account, and team
  • Click on any report card to expand and view details
  • Use "Full Details" link for the complete report page
  • Export to CSV, Excel, or PDF using the export buttons

Draft handovers can be edited anytime before submission:

  1. Go to Reports → Handover Reports
  2. Filter by status "Draft" if needed
  3. Find your draft (marked with orange "Draft" badge)
  4. Click the "Edit Draft" button
  5. Make your changes and save or submit
Technical Questions

ShiftOps works best with modern browsers:

  • Google Chrome (Recommended) - Latest version
  • Microsoft Edge - Latest version
  • Mozilla Firefox - Latest version
  • Safari - Latest version (Mac)

For the best experience, keep your browser updated to the latest version.

Reports can be exported in multiple formats:

  1. Navigate to the Reports page
  2. Apply your desired filters (date range, team, etc.)
  3. Click the export button for your preferred format:
    • CSV: For data analysis in spreadsheets
    • Excel: Formatted with multiple sheets
    • PDF: Professional report format for sharing

Exports include all filtered data with incidents, key points, change info, and KB updates.

If handover emails aren't being delivered, check the following:

  • Check Email Monitoring: Admins can view email delivery status in Admin → Email Monitoring
  • Verify Recipients: Ensure team members have valid email addresses
  • Check Spam Folder: Emails may be filtered by corporate email policies
  • Email Configuration: Contact admin to verify SMTP settings are correct

Contact your admin if issues persist.

Roster & Shift Questions

Access roster management through Tools → Shift Roster:

  • View Roster: Select month/year to view current schedule
  • Team Availability: Check who's available on specific dates
  • Upload Roster: Use the upload feature with Excel templates
  • Edit Individual: Click on any cell to modify a single shift

To generate shift allowance reports:

  1. Go to Tools → Shift Roster
  2. Click the "Shift Allowance" button
  3. Select the month and year (including past periods)
  4. Choose the account and team
  5. Select RD (Rotating Duty) members for shift allowance calculation
  6. Mark any exception members if needed
  7. Click "Download Report" to get the Excel file
Account & Access Questions

To change your password:

  1. Click on your profile icon in the sidebar
  2. Select "My Profile"
  3. Scroll to the "Change Password" section
  4. Enter your current password and new password
  5. Click "Update Password"

Note: If you use SSO login, password is managed through your corporate account.

ShiftOps has four user roles with different permissions:

  • User: Create/view handovers, view reports for their team
  • Team Admin: All user permissions + manage team members, roster, escalation matrix
  • Account Admin: All team admin permissions + manage multiple teams, user accounts
  • Super Admin: Full system access, manage all accounts, system configuration

To reset your password:

  1. Go to the login page
  2. Click "Sign in with credentials"
  3. Click the "Forgot password?" link
  4. Enter your registered email address
  5. Check your email for reset instructions

If you use SSO, use your corporate password reset process instead.
