Key Data Flows

① User opens browser → Nginx (HTTPS) → Gunicorn → Flask route → Jinja2 template rendered → HTML response
② Handover form edit → SSE stream established → YJS CRDT syncs deltas → collab_sse.py broadcasts to all editors in session
③ Handover submitted → email_service → Celery task queued in Redis → Celery worker sends via Flask-Mail / SMTP
④ Celery Beat fires every 2 min → ctask_scheduler.py → ServiceNow REST API → incident records upserted in MySQL
⑤ Login → session_token written to User row → every subsequent request validate_session() checks token → admin can force-logout any user
⑥ Roster generate → api_generate_schedule → roster_scheduler_algo.py → ScheduledShift rows written → publish notification via Celery
⑦ Container start → start.sh → alembic upgrade head → Flask app boots → APScheduler starts background jobs → Gunicorn serves
⑧ GitLab CI merge to master → manual deploy trigger → SSH to GCP VM → git pull → docker-compose prod rebuild → health check
