## Screenshot Guidelines

### Image Format
- **Format:** PNG (preferred) or JPG
- **Resolution:** 1920x1080 or similar
- **Size:** Optimize images (under 500KB each)

### Best Practices
1. **Clean Browser:** Use incognito/private mode to avoid extensions
2. **Full Width:** Capture the full page content
3. **Actual Data:** Use realistic test data, not placeholder text
4. **Consistent Theme:** Keep the same browser zoom level
5. **No Personal Info:** Blur or redact any sensitive information
6. **Multiple Roles:** Capture screens while logged in as different roles

### Capturing Admin Screenshots

For admin screenshots, you'll need to:

1. **Super Admin Screenshots:**
   - Login as a super_admin user
   - Capture all system configuration pages
   - Show full access to all features

2. **Account Admin Screenshots:**
   - Login as an account_admin user
   - Show limited menu (no system config)
   - Capture account-level management

3. **Team Admin Screenshots:**
   - Login as a team_admin user
   - Show team-only access
   - Capture team management features

### Tools for Screenshots
- **Windows:** Win + Shift + S (Snipping Tool)
- **Mac:** Cmd + Shift + 4
- **Chrome:** DevTools > More tools > Screenshot
- **Extensions:** Full Page Screen Capture, Awesome Screenshot

1. Save all screenshots in this folder (`docs/screenshots/`)
2. Verify all filenames match exactly (case-sensitive)
3. The documentation will automatically reference these images
4. Test the documentation by viewing it in a Markdown viewer

## Tips for Efficient Capture

1. **Plan your sessions:**
   - Session 1: Login as regular user, capture user guide screenshots
   - Session 2: Login as team_admin, capture team admin screenshots
   - Session 3: Login as account_admin, capture account admin screenshots
   - Session 4: Login as super_admin, capture super admin screenshots

2. **Use test data:**
   - Create some test handovers
   - Add test incidents, key points
   - Populate the roster
   - Generate some notifications

3. **Batch similar screenshots:**
   - Capture all form screenshots together
   - Capture all list views together
   - Capture all configuration pages together
